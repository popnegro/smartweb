Recomendaciones pendientes

Acciones sugeridas para elevar la puntuación y fortalecer la operación del dominio.

Prioridad Acción Motivo
Media Activar DNSSEC en DonWeb/NIC Argentina. Primero se firma la zona DNS en DonWeb; luego se publica el DS correspondiente en NIC Argentina para cerrar la cadena de confianza. Protege la zona frente a manipulación de respuestas DNS.
Media Solicitar PTR personalizado: mail.metzer.com.ar -> 200.58.111.25. El PTR no se configura en la zona DNS del dominio, sino en la delegación inversa administrada por el proveedor de la IP. Mejora reputación y consistencia del correo saliente.
Baja Validar periódicamente SPF, DKIM y DMARC. Evita degradación por cambios futuros en proveedores de correo.
Baja Verificar compatibilidad IPv6 del sitio web. Confirma experiencia estable para usuarios con conectividad IPv6.
