Funcionamiento de MiTaxi

MiTaxi es una plataforma Web-as-a-Service (WaaS) diseñada para conectar pasajeros y centrales de taxi de Mendoza sin intermediarios ni comisiones abusivas.

El sistema utiliza una interfaz ligera que no requiere descarga, permitiendo al usuario calcular su tarifa basada en la ubicación GPS y despachar el pedido directamente al número de WhatsApp de la central.

// Endpoint de despacho dinámico const dispatchUrl = `https://wa.me/${centralNumber}?text=${encodeURIComponent(orderData)}`;

Soluciones para Centrales

Colapso de Líneas

Reduce el tráfico de llamadas en horas pico al automatizar la toma de datos (origen/destino) mediante la web.

Direcciones Ambiguas

Elimina el error humano en el despacho gracias a las coordenadas precisas de Google Maps.

Beneficios para el Usuario

  • Sin Descargas: Funciona directamente en el navegador del móvil, ahorrando datos y memoria.
  • Transparencia: El pasajero conoce el valor estimado del viaje antes de solicitar la unidad.

User Flow

Diseño de interacción optimizado para completarse en menos de 30 segundos.

Inicio → Geo-ubicación → Cálculo Tarifa → Confirmación → WhatsApp

Escalabilidad

Arquitectura basada en Firebase (Backend) y Cloudflare (Edge), permitiendo soportar miles de peticiones simultáneas sin degradación del servicio.

Requerimientos

  • Google Maps JavaScript API
  • Navegador compatible con Geolocation API
  • Conexión móvil 3G o superior
