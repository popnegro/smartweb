Introducción

LocalPay es una solución de cobro ágil diseñada para el comercio de cercanía, optimizando la validación de pagos digitales y la gestión operativa diaria.

Objetivos del Proyecto

Antifraude
Validación interna sin depender de la pantalla del cliente.
Velocidad
Feedback visual y sonoro en menos de 2 segundos.

Gestión de Roles y Permisos

El sistema adapta la interfaz dinámicamente según el nivel de responsabilidad del usuario logueado.

Lógica de Seguridad

Los permisos se validan en el Backend (Middleware) mediante tokens JWT. La ocultación de botones en el frontend es puramente estética para mejorar la UX.

Semáforo Real-time (Webhooks)

Utiliza la tecnología de Notificaciones IPN de Mercado Pago para disparar el cambio de estado visual.

Endpoint del Webhook
POST /api/webhooks/v1/payments
