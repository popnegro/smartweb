Nivel 1.1

Checkout mediante QR Dinámico

Elimina errores de tipeo y automatiza la conciliación vinculando cada pago a una referencia única de venta.

Lógica Técnica

Se utiliza el endpoint de Instore Orders. El sistema envía el monto y un external_reference (ID de tu DB). Mercado Pago devuelve un binario de imagen o un string base64 para el QR.

Impacto Negocio

Reduce el tiempo en caja de 45s a 15s. El cliente solo escanea y paga; el cajero no necesita pedir el comprobante físico.

Nivel 1.2

Semáforo Real-time (Webhooks)

Validación de pagos instantánea mediante sockets, brindando paz mental al comerciante.

Cuando Mercado Pago recibe el dinero, dispara una notificación (Webhook) a nuestro servidor. El sistema procesa la firma de seguridad y avisa a la interfaz del cajero en menos de 2 segundos.

Nota: Incluye feedback sonoro "Beep" para que el vendedor no dependa de mirar la pantalla.
