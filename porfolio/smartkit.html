La necesidad de un estándar.

En el desarrollo de productos digitales, la inconsistencia es el enemigo número uno de la performance y la experiencia de usuario. Smartkit nació como una respuesta técnica para centralizar decisiones de diseño, tokens de color y comportamientos interactivos bajo un único framework interno.

// Smartkit Component Logic

const Button = ({ variant = 'primary', size = 'md', children }) => {
  const baseStyles = "inline-flex items-center transition-all font-black";
  
  const variants = {
    primary: "bg-lime-300 text-neutral-950 hover:scale-95",
    outline: "border border-white/20 text-white hover:border-lime-300"
  };

  return (
    <button className={`${baseStyles} ${variants[variant]}`}>
      {children}
    </button>
  );
};
